== Description ==

Sign up for a <a href="http://www.careerjet.com/partners/?ak=2a32e360e51c0cdc89499ea4cb06a60f" target="_blank">free Careerjet affiliate account</a>. Then add affiliate job listings from Careerjet and get paid for clicks!

== Installation ==

1. Upload the `wp-careerjet-shortcode` folder to the `/wp-content/plugins/` directory
2. Active the WP Careerjet Shortcode plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the <strong>Settings -> WP Careerjet Shortcode</strong> menu that appears in your admin menu
